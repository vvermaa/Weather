# Weather App

A simple and responsive Weather Web Application that provides real-time weather information for any city using the OpenWeather API.

## Live Demo
https://weather-sepia-xi.vercel.app/

---

##  Features
-  Search weather by city name
-  Displays temperature, weather condition, humidity, and wind speed
-  Dynamic UI updates based on weather conditions
-  Real-time data fetching using API
-  Responsive design for different screen sizes

---

##  Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API
- Vercel (Deployment)

---

##  How It Works
- User enters a city name
- The app sends a request to the OpenWeather API
- Weather data is fetched asynchronously using `fetch`
- The UI updates dynamically with the received data
